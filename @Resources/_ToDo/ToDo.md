1. **OmniCal** — "Omni" (всеобъемлющий) + "Cal" (календарь). Универсальный календарь, поддерживающий все .ics форматы.
2. **iCSGrid** — "iCS" (.ics формат) + "Grid" (сетка). Сетка событий для .ics файлов.
3. **TimeSync** — "Time" (время) + "Sync" (синхронизация). Синхронизация времени и событий через .ics.
4. **UniTimeline** — "Uni" (универсальный) + "Timeline" (временная шкала). Универсальная временная шкала для календарей.
5. **CalBlocks** — "Cal" (календарь) + "Blocks" (блоки). Календарь с блочным отображением событий.
6. **iCalMesh** — "iCal" (iCalendar/.ics) + "Mesh" (сетка/сеть). Сетка событий для iCalendar.
7. **SyncScheduler** — "Sync" (синхронизация) + "Scheduler" (планировщик). Планировщик с синхронизацией .ics.
8. **GridEvents** — "Grid" (сетка) + "Events" (события). Сетка для отображения событий календаря.
9. **iCloudGantt** — "iCloud" (iCloud поддержка) + "Gantt" (диаграмма Ганта). Gantt-стиль календаря с iCloud/.ics.
10. **UniversalCal** — "Universal" (универсальный) + "Cal" (календарь). Универсальный календарь для всех .ics форматов.
